# Dashboard de Vendas em Excel

Este projeto tem como objetivo criar um dashboard de vendas a partir de uma base de dados, transformando dados brutos em informações visuais claras que auxiliem na análise de desempenho e na tomada de decisões.

## Base de Dados
O arquivo `base.xlsx` contém informações de vendas, como:
- Data da venda
- Produto
- Categoria
- Região
- Quantidade vendida
- Valor da venda

## Ferramentas Utilizadas
- LibreOffice Calc (Linux)
- Microsoft Excel (.xlsx)
- GitHub

## Dashboard
O dashboard apresenta:
- Faturamento total
- Quantidade total vendida
- Ticket médio
- Vendas por mês
- Vendas por categoria
- Vendas por região
- Vendas por vendedor
